# concord-controlpack-iso42001

ISO/IEC 42001 control pack for [Concord](https://github.com/concord-dev/concord).

## Install

```sh
concord controlpack install ghcr.io/concord-dev/concord-controlpack-iso42001:v0.1.0
```

## Evidence sources

This pack consumes evidence from the following Concord plugins:

github, mlflow
